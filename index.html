Featured Project

Showcasing applied computer science concepts through interactive development.

Path Visualizer

Tech Stack: React.js, React Hooks, Bootstrap, JavaScript, SCSS, HTML

I developed an interactive grid application designed to visualize pathfinding algorithms in real-time. The core feature highlights Dijkstra's Algorithm, dynamically calculating and animating the shortest path between a user-selected start and end node. The application allows users to draw customizable obstacles on the grid, forcing the algorithm to intelligently recalculate the optimal route on the fly. This project demonstrates a strong understanding of data structures, graph theory, and state management within a modern React environment.
