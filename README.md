# Personal Portfolio Website

Welcome to the repository for my personal portfolio website! This site serves as a central hub to showcase my professional experience, technical skills, and recent projects.

## 🚀 Overview

This website is designed to be a lightweight, fully responsive portfolio. It has been customized with a sleek dark theme and tailored branding to provide a clean, modern user experience across all devices.

## 🛠️ Technologies Used

* **HTML5 & CSS3:** Semantic markup and custom styling.
* **JavaScript (jQuery):** For interactive elements, responsive breakpoints, and smooth scrolling (`scrollex` & `scrolly`).
* **FontAwesome:** For scalable vector icons and social media links.

## 💻 Getting Started

This is a static website, meaning no complex build tools or server setups are required to view or run it locally.
